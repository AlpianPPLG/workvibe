## 📝 Overview

The Team Management application is built with:

- **Frontend**: Next.js 13+ with App Router
- **Styling**: Tailwind CSS
- **Authentication**: NextAuth.js
- **Database**: (e.g., MongoDB, PostgreSQL)
- **Type Safety**: TypeScript
